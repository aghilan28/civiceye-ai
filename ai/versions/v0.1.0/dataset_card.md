# CivicEye Pothole Dataset v0.1.0

## Purpose

Initial foundation dataset for pothole detection only.

## Status

Template version. Add real images, annotations, validation reports, and split summaries before training.

## Required Before Training

- positive pothole images
- negative clean-road and false-positive trap images
- annotation review
- validation report
- YOLO export

## Compatibility

Designed for YOLOv8 object detection.
